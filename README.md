## ✨ Overview

A modern, interactive portfolio website showcasing my journey as a **Junior Full-Stack Developer**. Built with **React**, **TypeScript**, and **TailwindCSS**, featuring smooth GSAP animations, a scroll-stacking project showcase, and an interactive tech stack visualization.

## 🎨 Features

- **Scroll-Stacking Projects** – Unique card stacking animation with GSAP ScrollTrigger
- **Story Timeline** – Zigzag card layout with:- Alternating left/right positioning, expandable lightbox modal with full story view, center-pinned header that stays fixed while story cards scroll over it
- **Recent Reads** – Curated collection of articles, books, and X posts
- **Fully Responsive** – Optimized for all devices from mobile to 4K displays
- **Smooth Animations** – Scroll-triggered reveals, hover effects, and parallax elements
